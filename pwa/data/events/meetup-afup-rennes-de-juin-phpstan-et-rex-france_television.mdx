# Meetup AFUP Rennes de Juin : PHPStan et REX France Télévision

Retrouvons-nous le mercredi 28 juin à 19h chez Additi pour un nouveau meetup avec deux speakers :

- Frédéric Bouchery, Open-Data Team Leader chez CCM Benchmark, pour une "PHPStan Odyssey" : tout le monde connaît PHPStan, mais n'en mesure pas toujours l'impact. Frédéric nous proposera une séance de live coding où il tentera de corriger des anomalies remontées par PHPStan sur un vieux projet PHP. Si vous aimez les meetups techniques, vous allez être servis !
- Albin Kester, Lead Developer chez Les-Tilleuls.coop, pour présenter un retour d'expérience : "France Télévision, quand API Platform tombe à PIC". Albin présentera l’architecture et l’utilisation des différentes API pour supporter la charge des sites d’actualités de France 3 Régions, le Portail des Outre-mer La 1ère et Franceinfo.

Nous profiterons de ce meetup pour élire les représentant.e.s de l'antenne AFUP Rennes, qui vont contribuer à l'animer pour l'année à venir.

Un apéro et des pizzas seront offerts après les talks.
